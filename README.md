# 🌍 Global Supply Chain Risk Map
Global supply chain risk analysis using Python, Plotly, and Tableau

## 📌 Overview
Analyzed 180,000+ supply chain records to identify high-risk countries using delivery delays and lead time variance.

## 🎯 Business Problem
Which countries and shipping modes create the highest supply chain risk?

## 🔍 Key Insights
- First Class shipping has highest delay risk
- Several low-volume countries show extreme volatility
- Lead time variance is a strong early warning signal

## 🛠️ Tools
Python | Pandas | Plotly | Tableau

## 📊 Dataset
DataCo Supply Chain Dataset (180K+ rows)
